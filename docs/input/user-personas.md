# ターゲットユーザー

## メインペルソナ
### 属性
- **職業**: 介護・福祉施設のスタッフ（介護士、ケアマネジャー、施設長など）
- **ITリテラシー**: 高くはない。日常的なスマホ利用はするが、業務システムの複雑な操作は苦手な傾向。
- **デバイス**:
  - **業務**: タブレット（CogEvo利用端末）
  - **個人**: スマートフォン（プライベート）

### 1日の流れと利用シーン
- 施設利用者のリハビリやレクリエーションとしてCogEvoを使用している。
- 準備中や実施中に「ログインできない」「画面が進まない」「結果の見方がわからない」といったトラブルが発生。
- 忙しい合間なので、マニュアルを熟読する時間はなく、手っ取り早く解決したい。

### 課題・ペイン
- 電話は繋がるまで待たされる、メールは返事が遅い。
- マニュアルはどこにあるかわからない、専門用語が多くてわかりにくい。
- エラーが出るとパニックになりやすい。

### 求めているもの
- 「どうすればいいか」を簡潔に、専門用語を使わずに教えてほしい。
- タブレットや手持ちのスマホからすぐにアクセスしたい。

## 非ターゲット
- **CogEvoの開発者・エンジニア**: 技術的な深掘りをするためのツールではない。
