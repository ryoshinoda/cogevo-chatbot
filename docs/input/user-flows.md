# ユーザーフロー

## 1. アクセス〜認証
1. ユーザーは、個人のスマホまたは業務タブレットでサポートサイトのURLにアクセスする（QRコード、製品ページリンク等）。
2. パスワード入力画面が表示される。
3. 施設共通のパスワードを入力し、「開始」ボタンを押す。

## 2. 問い合わせ（メインフロー）
1. チャット画面が開く。
2. **免責事項**（「AIが回答するため誤りがある可能性があります」）が表示される。
3. **入力方法を選択する**:
   - **A. テキスト入力**: 入力欄に質問をタイプして送信。
   - **B. 音声入力**: マイクボタンを押し、話しかけて送信。
   - **C. クイック質問**: 画面上の「ログインできない」「登録方法は？」ボタンをタップして送信。
4. AIが回答を生成し、表示する。
   - 専門用語を避け、ステップバイステップで手順を案内。

## 3. 解決/未解決フロー
### 解決した場合
- ユーザーは手順に従い操作を行う。
- 「解決しましたか？」の問いに対し「はい」を選ぶ、またはそのままブラウザを閉じる。

### 解決しない/AIが答えられない場合
- AIが「申し訳ありません、その質問にはお答えできません」または「解決しない場合はこちら」と回答。
- **問い合わせフォームへのリンク**が表示される。
- ユーザーはリンクをタップし、従来の問い合わせフロー（メール/電話）へ進む。
