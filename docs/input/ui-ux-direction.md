# UI/UX方針

## 1. 全体の方向性
### デザインテーマ
- **「清潔感」「信頼感」「親しみやすさ」**
- ITツールへの苦手意識を軽減する、シンプルで明るいデザイン。

### カラー方針
- **メインカラー**: **CogEvoブルー**（薄い青、明るいトーン）
- **アクセント**: 清潔感のある白、視認性の高い黒（文字）
- **背景**: 白または非常に薄いグレー/青（目に優しい色）

## 2. インターフェース設計
### チャット画面
- **シンプルさ最優先**: 画面の大部分をチャットエリアにする。
- **入力エリア**: 
  - 下部に固定。
  - テキストボックスの横に「マイクアイコン」を配置。
  - 入力エリアの上部またはチャット開始時に「よくある質問ボタン」を配置。

### フォント・文字サイズ
- **大きめ**: 高齢者施設のスタッフがタブレットで見ることを想定し、文字サイズは標準よりやや大きめに設定する。
- **行間**: 詰めすぎず、読みやすさを重視。

## 3. UX原則
### 迷わせない
- 画面上の要素を最小限にする（「戻る」「ホーム」などは極力減らす）。
- 「何ができるか」をクイック質問ボタンで明示する。

### 安心感を与える
- AIが考えている間は「回答を作成中...」などのインジケーターを表示する。
- 答えられない場合も突き放さず、次のアクション（フォームへ）を丁寧に案内する。
