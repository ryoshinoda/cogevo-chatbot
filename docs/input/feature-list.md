# 機能一覧

## 1. ユーザー機能
| 機能名 | 説明 | 優先度 | 備考 |
|--------|------|--------|------|
| **簡易認証** | パスワードのみによるアクセス制限 | 🔴 Must | セキュリティ担保 |
| **テキストチャット** | 自然言語による質問と回答 | 🔴 Must | OpenAI API利用 |
| **音声入力** | ブラウザ標準機能を用いた音声認識ボタン | 🔴 Must | Web Speech API想定 |
| **クイック質問** | 頻出質問（2-3個）のボタン表示とワンタップ送信 | 🔴 Must | 入力負荷軽減 |
| **免責事項表示** | AI回答のリスクをユーザーに伝える表示 | 🔴 Must | 初回アクセス時/常時 |
| **フォーム誘導** | 解決できない場合の問い合わせフォームリンク表示 | 🔴 Must | エスカレーション |

## 2. システム・管理機能
| 機能名 | 説明 | 優先度 | 備考 |
|--------|------|--------|------|
| **RAG構築** | PDF/テキスト/HP情報を検索可能な状態にする | 🔴 Must | 精度向上の鍵 |
| **プロンプト制御** | 「メンターの語り口」「事務的かつ親切」な回答生成 | 🔴 Must | 人格設定 |
| **会話履歴保持** | セッション内での文脈理解 | 🔴 Must | |
| **ログ保存** | ユーザーの質問とAIの回答ログを保存 | 🟡 Should | 今後の改善用 |

## 凡例
- 🔴 Must Have（MVP必須）
- 🟡 Should Have（余裕があれば）
- 🟢 Nice to Have（将来検討）
