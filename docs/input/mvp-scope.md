# MVP範囲（フェーズ1）

## MVP定義
### 目的
- 問い合わせ対応の工数削減効果を検証する。
- 現場スタッフ（IT苦手層）がAIチャットボットを受け入れられるか確認する。

### スコープ
- **提供形態**: Webブラウザでアクセスする独立したサポートサイト。
- **対象データ**: 既存の公開済みマニュアル、FAQ、HP情報。
- **機能**: 
  - 共通パスワード認証
  - テキスト/音声/クイック入力によるチャット
  - 問い合わせフォームへの誘導

### 除外するもの（フェーズ2以降）
- CogEvo本体（アプリ/管理画面）への組み込み。
- ユーザーごとのID認証・個別履歴管理。
- キャラクター（アバターなど）の表示・アニメーション。
- 複雑な対話シナリオ（こちらから聞き返して診断するなど）。

## タイムラインイメージ
1. **要件定義・設計**: 完了
2. **プロトタイプ開発**: RAG精度検証、UI実装
3. **社内テスト**: 社員による回答精度チェック
4. **限定公開**: 一部特約店や信頼できる施設でのテスト運用
5. **全体公開**: 製品ページへのリンク設置
