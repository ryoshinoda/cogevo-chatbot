# CogEvo 問い合わせ対応AI（フェーズ1）

## 概要
認知機能向上ツール「CogEvo」を利用する介護・福祉施設スタッフ向けの、自動問い合わせ対応AIチャットボット。
OpenAI APIを活用し、社内ドキュメント（マニュアル、FAQ、HP）を元に、24時間365日質問に回答する。

## 解決する課題
- **問い合わせ対応の工数削減**: 専門的な知識が必要な問い合わせのエスカレーションによる、社内エキスパートの業務圧迫（目詰まり）を解消する。
- **ユーザーの利便性向上**: 電話やメールの待ち時間をなくし、現場で即座に自己解決できる環境を提供する。

## ターゲットユーザー
- **CogEvoを利用する介護・福祉施設のスタッフ**
- ITリテラシーは高くなく、業務ではタブレットを利用することが多い。

## 主要機能
- **自動回答チャット**: 自然言語での質問に対し、社内ドキュメントに基づき回答。
- **簡易認証**: 施設共通のパスワード入力によるアクセス制限。
- **マルチモーダル入力**: キーボード入力に加え、音声入力（ブラウザ標準）とクイック質問ボタンに対応。
- **エスカレーション**: AIで解決できない場合、問い合わせフォームへ誘導。
