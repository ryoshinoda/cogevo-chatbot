# プロダクト要件

## 1. プロダクト概要
### プロダクト形態
- **Webアプリケーション（サポートサイト）**
- レスポンシブ対応（スマホ、タブレット、PC）
- CogEvo本体とは独立したURLで提供（フェーズ1）

### 動作環境
- ブラウザ（Chrome, Safari, Edge等）
- マイク権限（音声入力利用時）

## 2. コア体験
### コンセプト
- **「エンジニアの頭脳」と「メンターの語り口」**
- 正確な技術情報を持ちつつ、初心者に寄り添って噛み砕いて説明する。

### 振る舞いの方針
- **事務的・端的**: 無駄な雑談はせず、解決策を提示する。
- **親切な翻訳**: 専門用語は使わず、誰でもわかる言葉に変換する。
- **リスクヘッジ**: 常に「AIによる回答」であることを意識させる。

## 3. 主要要件
### 入力インターフェース
- **テキスト入力**: キーボードによる自由入力。
- **音声入力**: ブラウザ標準APIを使用した音声認識。マイクボタンで起動。
- **クイック質問**: 「ログインできない」など、頻出質問をタップで即送信。

### 出力インターフェース
- **チャット形式**: 吹き出しによる対話UI。
- **免責表示**: チャット開始時または常設で免責事項を表示。
- **誘導リンク**: 回答できない場合、問い合わせフォームへのリンクを表示。

### 問い合わせフォーム
- **新規開発**: シンプルなWebフォームを作成する。
- **入力項目**: 施設名、担当者名、連絡先（メール/電話）、問い合わせ内容。
- **送信先**: 指定のサポート担当メールアドレスへ通知。

### 認証・セキュリティ
- **共通パスワード認証**: サイトアクセス時にパスワード入力を求める（ID不要）。
- **データ分離**: 一般ユーザー向けのため、社外秘（開発内部資料など）はデータソースに含めない。
